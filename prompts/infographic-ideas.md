# Infographic Ideas Prompt

## Usage
ใช้ prompt นี้เพื่อให้ AI ช่วยสร้างไอเดียและ brief สำหรับ Infographic

## Prompt Template

```
คุณเป็น Visual Content Strategist ช่วยสร้างไอเดีย Infographic:

## Topic
[หัวข้อที่ต้องการทำ Infographic]

## Data/Information
- [ข้อมูล 1]
- [ข้อมูล 2]
- [ข้อมูล 3]

## Purpose
[educate / compare / explain process / show statistics / timeline]

## Target Audience
[ระบุกลุ่มเป้าหมาย]

## Style Preference
[minimal / colorful / corporate / playful / data-heavy]

## Format
[vertical / horizontal / square / carousel]

## Key Message
[ข้อความหลักที่ต้องการสื่อ]

## Requirements
- ออกแบบให้อ่านง่าย
- มี visual hierarchy ชัดเจน
- เหมาะกับการแชร์บน social media
- สื่อสาร key message ได้ชัด
```

## Example

```
คุณเป็น Visual Content Strategist ช่วยสร้างไอเดีย Infographic:

## Topic
วิธีประหยัดค่าไฟในหน้าร้อน

## Data/Information
- เครื่องปรับอากาศกินไฟ 40% ของค่าไฟทั้งหมด
- ตั้งอุณหภูมิ 26°C ประหยัดได้ 10%
- ล้างแอร์ทุก 3 เดือนช่วยประหยัดไฟ

## Purpose
educate

## Target Audience
คนทั่วไป ครอบครัว

## Style Preference
colorful, playful

## Format
vertical (สำหรับ Instagram/Facebook)

## Key Message
ประหยัดค่าไฟง่ายๆ ทำได้เลยวันนี้
```

## Tips
- ใช้ตัวเลขและ statistics ทำให้น่าสนใจ
- จำกัดข้อความ ใช้ visuals แทน
- คิดถึง platform ที่จะ post (IG, FB, Twitter)
- ทำให้ scannable - อ่านผ่านๆ ก็เข้าใจ
