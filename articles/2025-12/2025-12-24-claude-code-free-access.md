## Headline
Claude Code Now Free! AI Coding Tool from Anthropic

## Lead
Anthropic has made Claude Code available for free at modest usage levels. This AI coding tool works through Terminal and IDE, with capabilities to understand your codebase and assist coding tasks through natural language commands.

## Key Points
- Claude Code is free at modest usage levels
- Works via Terminal, IDE, or @claude tag on GitHub
- Powered by Claude 3.7 Sonnet
- Supports extended thinking mode for complex tasks

## Details
**Claude Code** is an agentic coding tool developed by Anthropic, designed to work in developers' terminals with key capabilities:

1. **Understands Codebase** - Analyzes and comprehends project structure
2. **Execute Tasks** - Runs commands and automates workflows
3. **Git Workflows** - Manages version control
4. **Natural Language** - Accepts commands in natural language

### Pricing & Access
- **Free Tier**: Free at modest usage levels
- **API Pricing**: $3/M input tokens, $15/M output tokens
- **Plans**: Supports Free, Pro, Team, and Enterprise

## Impact
- Developers can try AI coding assistant for free
- Reduces barriers to accessing AI tools for developers
- Adds options in the AI coding tools market
