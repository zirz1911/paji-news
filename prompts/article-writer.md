# Article Writer Prompt

## Usage
ใช้ prompt นี้เพื่อให้ AI ช่วยเขียนบทความข่าว

## Prompt Template

```
คุณเป็นนักเขียนข่าวมืออาชีพ ช่วยเขียนบทความข่าวตามข้อมูลต่อไปนี้:

## Topic
[ระบุหัวข้อข่าว]

## Type
[breaking-news / feature-article / summary]

## Tone
[formal / casual / informative / analytical]

## Target Audience
[ระบุกลุ่มเป้าหมาย]

## Key Information
- [ข้อมูลสำคัญ 1]
- [ข้อมูลสำคัญ 2]
- [ข้อมูลสำคัญ 3]

## Language
[Thai / English / Both]

## Length
[short: 200-300 words / medium: 400-600 words / long: 800+ words]

## Requirements
- เขียนตรงประเด็น กระชับ
- ใช้ภาษาเข้าใจง่าย
- มี headline ที่ดึงดูด
- มี lead ที่สรุปใจความสำคัญ
- จัดโครงสร้างชัดเจน
```

## Example

```
คุณเป็นนักเขียนข่าวมืออาชีพ ช่วยเขียนบทความข่าวตามข้อมูลต่อไปนี้:

## Topic
Apple เปิดตัว iPhone 16

## Type
breaking-news

## Tone
informative

## Target Audience
ผู้ใช้ทั่วไปที่สนใจเทคโนโลยี

## Key Information
- เปิดตัววันที่ 10 กันยายน 2024
- มี AI features ใหม่
- ราคาเริ่มต้น 32,900 บาท

## Language
Thai

## Length
medium
```

## Tips
- ยิ่งให้ข้อมูลละเอียด ผลลัพธ์ยิ่งดี
- ระบุ sources ถ้ามี
- บอก angle ที่ต้องการ (positive/negative/neutral)
